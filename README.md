# blogpost-crud-jsp.servlet
A simple blogposting application that manage a collection of users with basic feature like( insert,delete,update,create) using JSP Servlet,JDBC MYSQL, Apache TOMCAT,IDL Eclipse

Using TOOLs
* IDL-Eclipse 2023-06
*jsp-2.2+
* Apache Tomcat-10.1
*Servlet API-2.5
*JDK-1.8
*MYSQL-connector-java-8.jar

Development Steps
1.create a Dynamic web project on Eclipse
2.create a project Structure
3.Setup MySQL Database
4.create a home.java
5.create a crud.java
6.create a BlogServlet.java
7.create a list.jsp
8.create a form.jsp
9.create a Error.jsp
10.Testing the Application
